FormCrafts allows you to create gorgeous forms and manage submissions

== Description ==

A drag-and-drop form builder to create and embed forms, and track submissions.

Features:

1. Drag-and-drop builder
2. Multi-page forms
3. Conditional logic
4. SSL secured
5. Intuitive submissions manager
6. Autoresponders
7. Customizable email notifications
8. Auto-save form data (allows users to finish filling the form later)

== Installation ==

1. Install and activate the plugin
2. Click on FormCrafts on the WP menu, and follow the steps
